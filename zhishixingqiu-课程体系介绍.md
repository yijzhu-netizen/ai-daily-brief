# 知识星球「智造本质」课程体系

> 制造业管理体系与ISO认证实操课程。面向企业管理者、内审员、技术转管理人员。
> 两门课程：管理通识课 + 能源管理体系实操课。

---

## 课程一：《AI时代，你更应该学会管理》

**讲师：** 炖刀（10年制造业管理 + 10年ISO认证审核）
**课时：** 16讲 + 番外篇，约5-6小时阅读量
**定价：** 299元
**对标框架：** 罗宾斯《管理学》第13版，融合德鲁克、特劳特、稻盛和夫经典
**定位：** 零基础入门，不讲理论史和空洞概念，每讲一个工具学了就能用

### 模块结构

| 模块 | 讲数 | 内容 |
|:---|:---:|:---|
| 模块一：入门——管理是什么 | 第1-2讲 | 管理定义、管理史、中国管理现实 |
| 模块二：P（计划）——定方向做决策 | 第3-5讲 | 计划、决策、变革与创新 |
| 模块三：D（执行）——搭班子带队伍 | 第6-13讲 | 组织结构、人力资源、团队、战略、领导、激励、沟通、情绪智力 |
| 模块四：C（控制）——管过程看结果 | 第14讲 | 控制机制、过程管理 |
| 模块五：A（改进+收口） | 第15-16讲 | 运营质量、持续改进、PDCA地图 |
| 番外篇 | 1讲 | 用管理方法论搭AI系统（Gate/Sage实战拆解） |

### 核心亮点

1. **AI时代差异化**：每讲每节扣住「AI来了，管理是过时了，还是更重要了？」，结论：AI能辅助执行（D），不能替代计划（P）和检查（C）
2. **制造业真实案例**：苏州上市公司备库存锁价、常州口罩公司熔喷布亏损、电装车间换模从24小时到5分钟、皮带一裁为二成本减半寿命翻倍、检测室取料流程创新等
3. **方法论内核「管P和C，不盯D」**：管理者聚焦计划和检查，执行交给团队和AI
4. **番外篇实战拆解**：完整展示用管理方法论设计AI系统（Gate审核助手 + Sage创作研发助手）的全过程

### 每讲结构（8环节）

自我介绍 → 反常识开场 → 本讲学习目标（含AI时代视角） → 核心内容（理论+大白话+真实案例+AI启示） → 实操工具（表格/模板） → 经典观点（德鲁克/稻盛/特劳特引用） → 本讲小结 → 课后作业

## 课程二：《能源管理体系：39节落地实操课》

**讲师：** 炖刀（10年制造业管理 + 10年ISO认证审核）
**课时：** 39课
**覆盖标准：** GB/T 23331-2020（等同ISO 50001:2018）
**配套法规：** 节约能源法、重点用能单位节能管理办法、GB/T 2589、GB/T 17167等

### 模块结构

| 模块 | 课数 | 内容 |
|:---|:---:|:---|
| 模块一：基础知识 | 第1-5课 | 能源管理体系概念、ISO 50001框架、与其他体系融合、术语定义、统计计量基础 |
| 模块二：标准条款解读 | 第6-17课 | GB/T 23331-2020第4-10章逐条拆解（标准原文+实施要点+真实案例） |
| 模块三：配套国标 | 第18-23课 | 6个核心国标实操解读（GB/T 2589、GB/T 17167、GB/T 39775等） |
| 模块四：政策与限额 | 第24-27课 | 节约能源法、双碳政策、61项能耗限额强制国标速查 |
| 模块五：行业应用 | 第28-33课 | 钢铁、化工、建材、纺织、有色、造纸/电力/石化8大行业实战 |
| 模块六：审核实务 | 第34-37课 | 内部审核全流程（策划→检查表→现场实施→不符合项→整改验证） |
| 模块七：认证与资质 | 第38-39课 | 能源管理体系认证规则、审核员注册准则 |

### 关键条款覆盖

| 标准条款 | 课程 | 核心内容 |
|:---|:---|:---|
| 6.3 能源评审 | 第9课 | 完整能源评审报告示例，SEU识别方法 |
| 6.4-6.6 EnPI/EnB/数据收集 | 第10课 | 归一化能耗手算方法，绩效参数设定 |
| 6.2 目标指标与管理方案 | 第11课 | 目标与指标区分，SMART原则应用 |
| 9.1 监视测量 | 第15课 | 合规性评价，数据真实性验证 |
| 9.2-9.3 内审与管理评审 | 第16课 | 内审程序、管理评审输入输出 |

### 行业覆盖

| 行业 | 限额标准 | 碳市场进度 |
|:---|:---|:---:|
| 钢铁 | GB 21256-2025 | 2025年已入市 |
| 化工（氯碱/电石/纯碱） | GB 31825等 | 2027年入市 |
| 建材（水泥/玻璃/陶瓷） | GB 16780等 | 2025年已入市 |
| 纺织印染 | GB 4287等 | — |
| 有色金属（铝/铜） | GB 21346等 | 2027年入市 |
| 制浆造纸/电力/石化 | GB 31825等 | 造纸2027年入市 |

### 课程特色

- **标准覆盖**：GB/T 23331-2020第4-10章全覆盖，条款号经核对不混淆
- **法规配套**：节能法、管理办法、双碳政策、61项限额标准速查表
- **案例源自实务**：所有案例改编自真实审核经历，有具体工厂背景和数据
- **能碳协同**：各行业碳排放现状和碳市场进度单独成节
- **内审实务**：从策划到整改全流程，对照GB/T 19011-2021编写
