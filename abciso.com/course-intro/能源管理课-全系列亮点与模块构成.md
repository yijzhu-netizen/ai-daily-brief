## 模块一：基础知识（第1-5课）

**定位**：零基础入门，不讲废话，直接建立能源管理体系的认知框架。

| --- | -------------------- | ------------------------------- |
| 第1课 | 能源管理体系核心概念与发展历程      | PDCA循环、体系发展脉络、与其他管理体系的区别        |
| 第2课 | ISO 50001:2018标准框架总览 | 第4-10章全景图、条款之间的逻辑关系             |
| 第3课 | 能源管理体系与其他管理体系的融合     | 与ISO 14001/ISO 45001的整合策略、一体化审核 |
| 第4课 | 能源管理相关术语与定义          | SEU、EnPI、EnB等核心术语的准确理解          |
| 第5课 | 能源统计与计量基础            | 综合能耗计算原则、数据采集方法                 |

> 💡 **亮点**：第2课给出一张"条款关系地图"，初学者看完就知道标准从哪入手。

## 模块五：行业应用（第28-33课）

**定位**：6个高耗能行业的能源管理实战，对标限额标准，侧重能碳协同。

| 课号   | 标题             | 覆盖行业     |
| ---- | -------------- | -------- |
| 第28课 | 钢铁行业能源管理体系建设实践 | 钢铁       |
| 第29课 | 化工行业（氯碱、电石、纯碱） | 化工       |
| 第30课 | 建材行业（水泥、玻璃、陶瓷） | 建材       |
| 第31课 | 纺织印染行业         | 纺织       |
| 第32课 | 有色金属（铝、铜）      | 有色       |
| 第33课 | 制浆造纸、电力、石油化工   | 造纸/电力/石化 |

**每课的固定结构**：

* 行业能源特点 → 限额标准表格 → SEU识别 → 基准设定特殊变量 → 实施重点 → 碳排放能碳协同 → 实务案例

> 💡 **亮点**：每课对比新旧国标版本差异（如钢铁GB 21256-2025转炉3级从-10收严到-15）、按工序给出限额等级表、单独一节讲碳排放（碳市场覆盖进度：钢铁2025年已入市，化工2027年，造纸2027年）。33课含三个子行业。

## 课程特色总结

| 维度         | 特色                                       |
| ---------- | ---------------------------------------- |
| **标准覆盖**   | GB/T 23331-2020第4-10章全覆盖，部分重点条款配原文+要点+案例 |
| **法规配套**   | 节能法、管理办法、双碳政策、限额标准61项速查                  |
| **行业覆盖**   | 钢铁、化工、建材、纺织、有色、造纸、电力、石化8大行业              |
| **条款准确**   | 每课的条款号均经核对，能源评审6.3、绩效参数6.4、能源基准6.5不混淆    |
| **数据可溯**   | 限额标准值、法规要求、注册条件全部来自原文件，非经验推测             |
| **案例源自实务** | 所有案例改编自真实审核经历，不编造、不夸张                    |
| **能碳协同**   | 第28-33课各行业碳排放现状和碳市场进度单独成节                |
| **内审实务**   | 从策划到整改全流程，强调EnMS特有的审核深度                  |
